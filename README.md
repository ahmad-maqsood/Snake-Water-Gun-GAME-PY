# Snake Water Gun (Python)

A simple command-line implementation of the **Snake, Water, Gun** game, built in Python as a practice project.

This project focuses on basic game logic, user input handling, and control flow, making it suitable for beginners learning Python fundamentals.

---

## Game Rules

- Snake drinks Water → **Snake wins**
- Water disables Gun → **Water wins**
- Gun kills Snake → **Gun wins**

The game is a variation of Rock, Paper, Scissors.

---

## Features

- Command-line based interactive gameplay
- Randomized computer choices
- Score tracking across multiple rounds
- Input validation for player moves
- Option to continue or exit after each round

---

## How to Play

1. Run the Python script
2. Enter your choice when prompted:
   - `s` for Snake  
   - `w` for Water  
   - `g` for Gun
3. The computer selects its move
4. The winner is decided based on the rules
5. Continue playing or exit when prompted
